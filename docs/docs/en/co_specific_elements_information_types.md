# Which elements make up the information types?

Task topics, concept topics and reference topics each have specific XML elements in their body part:

|Task topic|Concept topic|Reference topic|
|----------|-------------|---------------|
|In `taskbody`:-   `context`
-   `prereq`
-   `steps`
-   `steps-unordered`
-   `step`

In `steps`:-   `cmd`
-   `substeps`
-   `choices`
-   `choicetable`
-   `stepxmp`
-   `stepresult`
-   `info`
-   `note`

On their own:-   `example`
-   `result`
-   `postreq`

|In `conbody`:-   `section`
-   `p`
-   `ul`, `li`
-   `ol`, `li`
-   `table`
-   `fig`, `image`
-   `example`

|In `refbody`:-   `section`
-   `p`
-   `properties`
-   `ul`, `li`
-   `ol`, `li`
-   `table`
-   `fig`, `image`
-   `example`

|
