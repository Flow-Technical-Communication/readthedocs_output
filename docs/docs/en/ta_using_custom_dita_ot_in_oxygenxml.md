# Using custom DITA-OT in Oxygen XML

1.  In Oxygen XML, choose **Options** \> **Preferences**.

2.  Go to the **DITA** page.

3.  Choose **Custom** in the **DITA Open Toolkit** section.

4.  Add the link to the correct DITA-OT.

5.  Click **Apply** \> **OK**.
